## 一、老韩分析体系核心框架

### 分析步骤（从105期视频中归纳的固定套路）

```
1. 日主五行+阴阳 → 确定"我是谁"
2. 身旺身弱 → 确定喜忌方向
3. 十神组合 → 确定性格底色和行为模式
4. 大运天干"透出" → 确定人生阶段主题
5. 流年验证 → 用已知事件校验盘的准确性
6. 综合画像 → 性格+事业+感情+财富+健康
```

### 核心公式

老韩的判断逻辑可以归纳为一个核心公式：

```
人生事件 = 原局结构 × 大运激活 × 流年触发
```

- **原局结构**：决定你"有什么能力/倾向"
- **大运激活**：决定"什么时候这个能力被打开"
- **流年触发**：决定"具体哪一年发生"

## 三、天干"透出"——最核心的判断信号（518次提及）

### 3.1 透出的核心规则

| 规则 | 老韩原文 | 引擎应用 |
|------|---------|----------|
| 天干透出 > 地支暗藏 | "大运中土透出的时候是上线最高的" | 大运分析权重：天干0.6 地支0.4 |
| 用神透出 = 好事在明面 | "水透出的时候=财运来了" | 大运天干为喜用 → 前5年好 |
| 忌神透出 = 问题在明面 | "土透干的时候=比劫压力期" | 大运天干为忌 → 前5年压力 |
| 透出时间 = 事件发生时间 | "1213水透出的时候（赚到钱）" | 流年天干透出与大运同步 = 爆发 |

### 3.2 透出→事件的判断模式

从105期视频中反复出现的判断公式：

```
当[用神五行]在大运天干透出 → 对应十神含义的好事发生
当[忌神五行]在大运天干透出 → 对应十神含义的坏事发生
当[大运透出]+[流年同五行] → 事件加倍放大
```

**实例**：
- "火在天干透出的时候是巅峰期" → 火为喜用(食伤)时
- "1415年食伤透出" → 作品爆发
- "水透出金水对他消耗没有关系" → 水为喜用时泄秀无害
- "前面1819年土透干的时候" → 比劫运压力期

### 3.3 大运前后半段

```
大运天干 = 前5年主题（明面）
大运地支 = 后5年主题（底层环境）
```

但老韩在实际分析中更侧重**天干**，地支更多作为"根气"辅助判断。

## 四、格局与用神判断

### 4.1 取用神的优先级

从105期视频中归纳老韩的取用神逻辑：

1. **先看月令（季节）**→ 确定五行旺衰基调
2. **再看整体五行力量对比** → 身旺/身弱
3. **身旺取用**：食伤泄秀 > 财星耗身 > 官杀制身
4. **身弱取用**：印星生身 > 比劫帮身
5. **特殊情况**：从格（五行极度偏旺时顺势而为）

### 4.2 从格判断

老韩在105期中仅提到从格5次，且多为否定：
- "有的人说这个盘是从格，他说不通的"
- "很多人会强行解释可能说他是从格"

**结论**：老韩倾向于正格论命，从格很少使用。

### 4.3 五行流通论（周深案例）

老韩在周深那期明确提出：
- "他这个牌最好的逻辑在于五行流通"
- "身弱不帮身也能很好"
- "关键不是旺衰，而是流通"

**五行流通判断**：
```
好的流通：日主 → 食伤 → 财星（能量顺畅流动）
差的流通：日主 ← 印星 ← 官杀（能量堵塞回流）
最好的配置：有泄有生有制有化，形成循环
```

## 五、性格分析法

### 5.1 性格判断的核心入口

老韩原话："八字决定性格，性格决定命运"

性格判断的三个入口：
1. **日主本性**（最基础）
2. **十神组合**（最具体）
3. **时柱十神关系**（"通过时辰关系直接去断性格比较快"）

### 5.2 十神→性格特征

### 5.3 "象法"看盘（单依纯案例）

事业爆发 = 用神在大运天干透出 + 流年配合

## 八、大运转折点判断（524次提及"大运"）

### 8.1 转折点类型

| 转折类型 | 判断条件 | 视频原文 |
|---------|---------|----------|
| 用神透出 | 大运天干为喜用 | "火在天干透出的时候是巅峰期" |
| 五行切换 | 从一种五行运切到另一种 | "真正比较不利的大运其实就是壬戌" |
| 新运起始 | 换运=换主题 | "在转这个伤官透出的大运之前" |
| 合/冲激活 | 大运地支冲动原局 | "正好壬申，这个金就是冲这个木的" |

### 8.2 "韩式反推法"

胡歌案例中老韩明确说"这就是经典的韩式反推法"：

```
已知事件 → 检查对应年份的大运+流年 → 找到命理逻辑 → 推导未来
```

步骤：
1. 找到命主人生中的已知转折点（成名/结婚/离婚/事业高峰）
2. 对应到那个年份的大运天干和流年天干
3. 确认"是哪个五行透出导致了这个事件"
4. 推导："同样的五行再次透出时，类似事件会重现"

### 9.3 风格融合指南

**老韩的表达风格特点**：
1. 直接、口语化、不绕弯
2. 具象化——不说"食伤旺"，说"靠说话吃饭的盘"
3. 因果链明确——"A所以B，因为C"
4. 用动词而非形容词——"他在打破秩序"而非"有叛逆性格"
5. 结合实际场景——"所以他是主持人嘛"

**引擎输出应该**：
- 先给结论（动词化描述）
- 再给逻辑（因为XX所以XX）
- 最后给场景（适合做XX、在XX时候表现为XX）

**示例对比**：

当前引擎输出：
> "食伤旺：表达欲强，有创造力和审美力，追求品质。"

融合后应该输出：
> "你是靠输出吃饭的人——不管是写东西、说话、做产品还是创作，把脑子里的东西推到外面才是你的核心通道。食伤对你而言=作品能力+表达欲+消耗品。走到食伤透出的大运就是你出成绩的巅峰期。"
