# 分析框架 v1 改进计划

> 基于50期视频深度提炼，2026-05-02

## 核心发现

从50期视频（362K字）中提炼出老韩分析的**5个核心方法论差异**（与我们当前引擎对比）：

| # | 老韩的做法 | 当前引擎 | 差距 |
|---|-----------|---------|------|
| 1 | 十神组合→动词化行为描述 | 统计个数→静态标签 | 大 |
| 2 | 大运天干透出=关键信号 | 不区分天干地支力量 | 大 |
| 3 | 五行流通判断（金水逻辑/火土逻辑） | 只算五行总分 | 中 |
| 4 | 日支=配偶宫+内心世界专项分析 | 只作为四柱之一展示 | 中 |
| 5 | 用经历反推验证（韩式反推法） | 无验证机制 | 小（留给AI问询）|

## 实施方案

### Phase 1: 十神组合检测引擎（P0）

**文件**: `src/combinations.ts`（新建）

新增十神组合检测模块，识别以下8种核心组合：

```typescript
interface TenGodCombination {
  type: string;           // 组合名称
  detected: boolean;      // 是否存在
  strength: number;       // 力度(0-10)
  personality: string;    // 性格描述（动词化）
  career: string;         // 事业影响
  relationship: string;   // 感情影响
  risk: string;           // 风险提示
}
```

**8种核心组合**：

| 组合 | 检测条件 | 动词化描述 |
|------|---------|------------|
| 食伤泄秀 | 身旺+食伤有力 | "你天生需要向外输出，创作和表达是你释放能量的通道" |
| 食伤生财 | 食伤+财星连续 | "你适合通过才华/作品变现，站着把钱赚了" |
| 伤官配印 | 伤官+印星同在 | "有才但有底线，清高而不空洞" |
| 伤官见官 | 伤官+正官同在 | "内心的叛逆和外部的规则在拉扯，容易冒犯权威" |
| 食神制杀 | 食神+七杀同在 | "用表达和才华化解压力，越有压力越出彩" |
| 杀印相生 | 七杀+印星连续 | "压力被转化为成长动力，逆境出成绩" |
| 比劫夺财 | 比劫旺+财星 | "有竞争者在争你的资源，合伙需谨慎" |
| 官杀混杂 | 正官+七杀同在 | "两种约束力交叉，做事容易犹豫" |

**检测逻辑**：
- 不是简单看"有没有"，而是看"力度够不够"
- 考虑透干/暗藏的区别
- 考虑身旺身弱对组合效果的影响

### Phase 2: 大运天干/地支分层分析（P0）

**文件**: `src/analysis.ts` → `buildFlowAnalysis` 修改

当前：大运干支作为整体分析
改为：
- 天干 = 前5年主题，力度权重 0.6
- 地支 = 后5年主题，力度权重 0.4
- 天干透出用神 → 明面机会
- 天干透出忌神 → 明面问题
- 地支暗藏 → 底层环境

增加判断：
```typescript
interface LuckCycleLayered {
  stemAnalysis: string;   // 天干分析（前5年）
  branchAnalysis: string; // 地支分析（后5年）
  stemIsPositive: boolean;
  branchIsPositive: boolean;
  transitionYear: number; // 转折年（约大运中点）
}
```

### Phase 3: 五行流通判断（P1）

**文件**: `src/analysis.ts` 新增 `analyzeElementFlow`

老韩反复提到的"逻辑"：
- "金水逻辑" = 食伤生财的完整链路（身旺→金泄土→水为财）
- "火土逻辑" = 印生比劫（火生土）— 对身旺是坏事
- "木火逻辑" = 官杀+印（木克土+火生土）

检测五行是否形成连续生克链：
```
日主 → 食伤(泄) → 财星(生) = 流通好
日主 ← 印星(生) ← 官杀(生印) = 可能过旺
```

### Phase 4: 日支专项深度分析（P1）

**文件**: `src/analysis.ts` 新增 `analyzeDayBranch`

- 日支十神含义（配偶宫）
- 日支地势含义（内在能量状态）
- 日支是否为阳刃（午对戊、卯对甲等）
- 日支逢冲/合/刑的影响
- 日支与大运地支的互动

### Phase 5: 性格分析动词化升级（P0）

**文件**: `web/src/renderer.ts` → `renderPersonality` 重写

从 "你有食伤旺" → "你天生需要向外输出和表达"

每个十神组合输出：
- **行为动词**：你会怎么做
- **动机解释**：为什么这么做
- **场景具象**：在什么场景下最明显
- **潜在风险**：过度时会怎样

### Phase 6: 感情时间窗口（P1）

**文件**: `src/analysis.ts` 新增 `predictRelationshipWindows`

基于：
- 男命：偏财/正财透出年 = 感情活跃
- 女命：正官/七杀透出年 = 感情活跃
- 桃花星逢冲年 = 感情变动
- 合到日支年 = 有人进入生活
